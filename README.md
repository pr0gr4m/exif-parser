exif-parser
============

# Enviornment

* OS : Ubuntu 16.04.3 64 bit
* Kernel : 4.13.0-26-generic
* IDE : Qt Creator

# Dependency

* qt5

```
$ sudo apt install qt5-default qt5-qmake
$ sudo apt install g++ binutils make build-essential
```

# Build

```
$ cd src
$ mkdir build
$ cd build
$ qmake ../
$ make
```
